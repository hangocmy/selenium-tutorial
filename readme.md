# HOW TO RUN
## Install Python and Selenium:
  - Install Python.

  - Install Selenium: pip install Selenium.


## Install Driver:

  - Download WebDriver: Chrowe Driver, Firefox Driver, Safari Driver, Edge Driver,...
	
  - Install WebDriver Manager: pip install webdriver_manager.


## Run the script:

  > `cd tests`.

  > `python tên_file.py`.
